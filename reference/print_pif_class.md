# Print a `pif_class`

Prints a `pif_class` object.

## Usage

``` r
print_pif_class(x, accuracy)
```

## Arguments

- x:

  A `pif_class`

- accuracy:

  The accuracy parameter for
  [`scales::percent`](https://scales.r-lib.org/reference/percent_format.html).

## Value

Called for its side-effects of printing to the console
