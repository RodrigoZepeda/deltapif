# Print a `cases_class`

Prints a `cases_class` object.

## Usage

``` r
print_cases_class(x, accuracy)
```

## Arguments

- x:

  A `cases_class`

- accuracy:

  The accuracy parameter for
  [`scales::comma`](https://scales.r-lib.org/reference/comma.html).

## Value

Called for its side-effects of printing to the console
